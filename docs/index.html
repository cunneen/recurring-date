Recurring Date - Playground

@cunneen/recurring-date is a small, zero-dependency, Javascript utility for generating recurring dates i.e. dates that repeat on a regular schedule (e.g. "Every 3 days", or "Every third Tuesday and Thursday").

See below for instructions and an interactive demonstration.

See the GitHub repo for more details.
